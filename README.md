# Python car game 
This is a simple command-line car game built while following Programming with Mosh's Python course.

In this project I was able to practice conditions, loops and user input handling.

## Features
- Starting the car
- Stopping the car
- Help menu
- Quit command
- It also contains state checks and guard conditions

## How to run
Install Python
Download the project
Run:

```bash
python car game.py
```

## What I Learned
- I was able to use my learning on if/else and elif statemtents
- Manage program states through the use of boolean variables
- Handling user input

## Challenges I faced

- When it came to the code recognising that the car was already started or stopped. It kept printing both statments for when you first turn the car on/ off and when the car was already running/off
- Struggled a lot with using boolean to return the correct statment or anything at all at times
