# Budgeting App
This is my first independent Python project, made without using tutorials, only using Google for debugging

It allows users to track income, expenses and view monthly summaries of their finance.

This was created as part of my journey to improve my understanding of Python, especially in the finance sector of Python and maths.

## Features
- Add income to monthly budget
- Add expenses in categories:
  - Entertainment
  - Food
  - Transport
  - Living costs
- Automatic budget updates after each entry
-  Warnings when a category becomes too high
- Time import to see goodbye message
- Exit option to close the program


## How to run
Install Python

Download the project

Run the program with:
````bash
BeginnerBudgetingApp.py
````
